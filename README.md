# Проект Mesto (React)

## О проекте:
Мы разработали функциональный сайт, где пользователи могут делиться фотографиями, добавлять свои и ставить лайки.

## Технологии, используемые в проекте:
1. React 
2. JSX
3. Хуки useState и useEffect
4. Функциональные компоненты
